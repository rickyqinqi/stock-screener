# 深证A股 · 均线多头排列筛选器

每日自动筛选深证A股中**均线多头排列**且满足财务条件的股票，结果通过 GitHub Pages 展示。

## 筛选条件

| 条件 | 说明 |
|------|------|
| 均线多头排列 | MA 多头发散策略（MA5 > MA10 > MA20 > MA60） |
| 归母净利润 > 0 | 归属于母公司所有者的净利润为正 |
| 3月涨幅 0%~100% | 最近60个交易日涨跌幅在 0% 到 100% 之间 |
| 深证A股 | 仅限深圳证券交易所 A 股 |

## 结果展示

- **Web 页面**: `https://<用户名>.github.io/<仓库名>/`
- **数据文件**: `docs/data/latest.json`
- **历史记录**: `docs/data/history/YYYY-MM-DD.json`

## 自动刷新

- 每个工作日北京时间 18:00 自动运行
- 手动触发：GitHub Actions → `Daily Stock Screener` → `Run workflow`

## 本地运行

```bash
# 前置要求：Node.js >= 18

# 运行当天筛选（使用昨天日期）
node scripts/screen.js

# 指定日期
node scripts/screen.js --date 2026-06-17
```

## 数据来源

数据通过腾讯自选股公开数据接口获取，仅供参考。
